## How to Use

### Download Manually

```bash
npx degit graphcms/graphcms-examples/using-rich-text-react-renderer using-rich-text-react-renderer
```

Install & Run:

```bash
cd using-rich-text-react-renderer
npm install
npm run dev
# or
cd using-rich-text-react-renderer
yarn
yarn dev
```
